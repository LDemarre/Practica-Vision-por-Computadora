## **Introducción**
En este repositorio se subirán todos los ejercicios **obligatorios** procedentes de la práctica de cada unidad dentro de la asignatura. Cada unidad esta dividida en una carpeta donde se encontrarán los ejercicios de dicha unidad, cada ejercicio a su vez también tiene su propia carpeta que contendra todos los recursos utilizados para la realización de dicho ejercicio.
